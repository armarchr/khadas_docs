title: Khadas Settings
---

This guide describes settings that are unique to Khadas SBCs.

## Locating Khadas settings

`Settings-->Khadas settings`
<img src="/android/images/vim4/KhadasSettings1.png" width="75%" height="75%">

## Menu descriptions

### HDMI-in
Display video stream from HDMI input.

### Ethernet
Display ethernet settings.

### Display & sound 
Backlight adjustment, screen orientation and resolution.
<img src="/android/images/vim4/KhadasSettings2.png" width="75%" height="75%">

### Power key definition
The power key can function as a sleep, shutdown or restart button.
<img src="/android/images/vim4/KhadasSettings3.png" width="75%" height="75%">

### Function key definition
By default, the function key is defined as HOME. A new value can be assigned from the Android key code values table.
<img src="/android/images/vim4/KhadasSettings4.png" width="75%" height="75%">

### LED behaviour
Control the behavior of the white and red LEDs.
<img src="/android/images/vim4/KhadasSettings5.png" width="75%" height="75%">

### Cooling fan
Adjust the fan speed manually. By default, it is automatically adjusted according to the temperature.
<img src="/android/images/vim4/KhadasSettings6.png" width="75%" height="75%">

### WOL
Wake on LAN (WOL) allows you to wake up your SBC through a LAN network with your mobile phone or computer. Please refer to [WOL](Wol.html).

### Status bar visibility
Hide or display of the top and bottom status bars.
<img src="/android/images/vim4/KhadasSettings7.png" width="75%" height="75%">

### Logo LED control
Enable / disable the logo LED lights.
<img src="/android/images/vim4/KhadasSettings8.png" width="75%" height="75%">

### Timer
Define fixed timings for shutdown and startup.
<img src="/android/images/vim4/KhadasSettings9.png" width="75%" height="75%">

### OTA Updates
Enable/disable over-the-air updates for Android.
